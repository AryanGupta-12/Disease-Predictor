# Disease Predictor

Welcome to the Disease Predictor application.

## Key Strengths of the Disease Predictor

This application offers several notable strengths that enhance its usability, predictive accuracy, and maintainability. For a detailed analysis, please refer to the [disease_predictor_strengths.md](docs/disease_predictor_strengths.md) document.

- Robust preprocessing using symptom severity weighting and scaling.
- Confidence-based prediction output.
- User-friendly, interactive Streamlit-based UI.
- Modular data management with external CSV datasets for symptoms, diseases, and precautions.
- Lightweight architecture for easy deployment and maintenance.
